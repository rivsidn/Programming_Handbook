## if 语句

### 数字比较

| 比较符号 | 说明     |
|----------|----------|
| -eq      | 相等     |
| -ne      | 不相等   |
| -gt      | 大于     |
| -lt      | 小于     |
| -ge      | 大于等于 |
| -le      | 小于等于 |

因为'>'、'<' 在shell 中有重定向的功能，所以不作为比较符号使用.

### 字符串比较

| 比较符号 | 说明               |
|----------|--------------------|
| -z       | 判断字符串是否为空 |
| -n       | 判断字符串是否非空 |
| =        | 判断字符串是否相等 |
| !=       | 判断字符串是否不等 |

#### 应用示例

```bash
if [ -z "$1" ]; then
    echo "null"
else
    echo "not null"
fi
```

由于这里的"$1" 可能为空，所以判断的时候需要加引号.


### 文件类型比较

| 比较符号 | 说明           |
|----------|----------------|
| -e       | 是否存在       |
| -f       | 是否为普通文件 |
| -d       | 是否为目录     |
| -r       | 是否可读       |
| -w       | 是否可写       |
| -x       | 是否可执行     |



## case 语句

```bash
#!/bin/bash

case "$1" in
  start)
    echo "启动服务"
    ;;
  stop)
    echo "停止服务"
    ;;
  restart)
    echo "重启服务"
    ;;
  *)
    echo "用法: $0 {start|stop|restart}"
    ;;
esac
```
