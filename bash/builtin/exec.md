

### 语法

```bash
exec [-cl] [-a name] [command [arguments]]
```



### 参数说明

| 参数    | 说明                                        |
| ------- | ------------------------------------------- |
| -l      | 给传递给`command`的第零个参数添加一个破折号 |
| -c      | 清空执行命令的环境变量                      |
| -a name | 将`name`作为第零个参数传递给`command`       |



###  使用说明

* 如果有命令，执行当前命令
* 如果没有命令，可以通过重定向影响当前`shell`的执行环境



### 使用示例

```bash
FIFO="test_fifo"

mkfifo ${FIFO}

# 以读写方式打开文件
exec 4<>${FIFO}
rm ${FIFO}

exec 4<&-	# 关闭描述符的读
exec 4>&-	# 关闭描述符的写

```
