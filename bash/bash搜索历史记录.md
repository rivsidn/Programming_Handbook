| 快捷键 | 解释                                     |
| ------ | ---------------------------------------- |
| C-r    | 开始搜索，继续搜索上个匹配命令           |
| C-j    | 结束搜索，保留当前搜索内容，可以继续编辑 |
| C-g    | 结束搜索，恢复之前行内容                 |
| RET    | 结束搜索，执行当前搜索获得的命令         |
