## 环境搭建

### 安装

```bash
# 安装tcl命令
sudo apt-get intall tcl
# 安装tcl man手册
sudo apt-get install tcl-doc
```

### 手册查询

```bash
# man手册查询，查询while命令
man 3tcl while
```



## 语言基础

### 注释

* 行首注释

  ```tcl
  # 这是一行注释
  ```

* 行中注释

  ```tcl
  puts "comment line" ;# 这也是一行注释
  ```

  注释不在行首需要紧跟着分号。
