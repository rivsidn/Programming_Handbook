## 交互解释器运行

```bash
$ python3
Python 3.8.10 (default, Nov 26 2021, 20:14:08) 
[GCC 9.3.0] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>> print("Hello, World!")
Hello, World!
>>>
```

结束之后，`quit()`  或者 `ctrl+D` 退出。

## 脚本运行

```python
#!/usr/bin/python

print("Hello, World!\n")
```

运行与`shell`脚本一致。
