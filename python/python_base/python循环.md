## while循环

```python
i = 0

while 1 < 7:
    i += 1
    if i == 3:
    	continue
	print(i)
```



### continue语句

停止当前迭代，继续下一个。

### break语句

通过`break`  语句，可以在循环遍历所有项目之前停止循环。 



## for循环

`for`循环用于迭代序列(列表、元组、字典、集合或字符串)。

```python
for x in range(10):
    print(x)

for x in range(10):
    print(x)
else
	print("Finally finished!")
```

**pass** 可以作为`for` 循环中的空语句。
