
## Go 变量

### 变量定义

```go
package main

import (
	"fmt"
)

func main() {
	var a int = 10; //标注写法
	var b int;      //b 自动初始化为 0
	c := 100;       //自动推理c 变量类型

	fmt.Println(a)
	fmt.Println(b)
	fmt.Println(c)
}
```

### 变量组

```go
var (
    a int
    b string
)
```

变量组定义，多用于全局变量.
