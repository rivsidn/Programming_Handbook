### 字符串操作符

```perl
# 拼接
$str1 = "hello ";
$str2 = "world\n";

print $str1.$str2;				# 输出 hello world

# 重复
$str = "fred";
print $str x 3;					# 输出fredfredfred

```



### 比较操作符

| 比较       | 数字 | 字符串 |
| ---------- | ---- | ------ |
| 相等       | ==   | eq     |
| 不等       | !=   | ne     |
| 大于       | <    | lt     |
| 小于       | >    | gt     |
| 小于或等于 | <=   | le     |
| 大于或等于 | >=   | ge     |
